# The famous o3 "GeoGuessr" prompt did not work
- URL: https://www.seangoedecke.com/the-o3-geoguessr-prompt-did-not-work/
- Added At: 2026-05-21 13:58:37
- Tags: #read #llm

## TL;DR
本文通过基准测试对比了OpenAI o3模型在地理定位任务中默认与复杂提示词的效果，发现默认提示词表现更优，复杂提示词无显著提升，表明过度工程化提示词可能无效。同时，GPT-5.4和GPT-5.5不具备o3的该能力，强调需严谨评估提示词效果。

## Summary
文章探讨了OpenAI o3模型在地理定位任务中的表现，以及所谓的“GeoGuessr提示词”是否真正有效。作者通过构建一个包含200张图片的基准测试，对比了默认提示词与复杂提示词的效果，发现默认提示词在多数指标上表现更优，复杂提示词并未带来显著提升。这揭示了在模型本身能力较强时，过度工程化的提示词可能只是自我安慰，而非真正提升性能。此外，作者还测试了GPT-5.4和GPT-5.5，发现它们不具备o3的地理定位能力，表明该特性并未在后续模型中延续。文章强调，评估提示词效果需要严谨的基准测试，而非仅凭主观感受。
