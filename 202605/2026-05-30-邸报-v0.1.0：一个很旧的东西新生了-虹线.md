## Summary
邸报 v0.1.0 是一个开源的 RSS 阅读器，旨在通过本地部署和算法推荐，帮助用户在不扩大信息来源的前提下，优化已订阅内容的阅读顺序。它运行在 NAS、VPS 或本地电脑上，支持导入 OPML 和添加 RSS 地址，具备阅读、收藏和标记功能。核心特点是基于用户行为学习，对文章进行重新排序，并提供可解释的推荐理由，而非黑盒评分。

项目强调数据自主性，所有数据存储在本地 SQLite 文件中，无需依赖中心化服务或 LLM API，仅需接入免费的 embedding 服务（如硅基流动）或本地运行小型模型即可实现推荐。邸报被描述为“外部嗅觉器官”，旨在辅助用户发现潜在感兴趣的内容，而非替代判断。

作者开发此项目的动机源于对当前平台算法主导信息分发的不满，以及 RSS 订阅在中文互联网中的衰落。邸报试图平衡信源自主与排序自动化，解决用户既厌倦平台推荐又不愿手动处理大量 RSS 订阅的痛点。

v0.1.0 版本已发布，支持 Docker Compose 一键部署，项目为 source-available 模式，鼓励用户通过 GitHub Star 和 Issues 参与反馈。代码由 AI 生成，体现了用新技术复兴旧事物的尝试。
