# The End of the Open Web

- Added At: 2026-05-02 14:14:09
- Tags: #read

## TL;DR
文章指出开放网络因平台垄断、AI整合及匿名性丧失而衰落，预测未来将更封闭并形成信息鸿沟。作者呼吁支持独立创作以延续开放精神。

## Summary
文章《开放网络的终结》探讨了开放网络（Open Web）的衰落及其未来可能走向封闭的趋势。作者回顾了早期互联网的开放与自由，指出如今网络正经历“恶化”（enshittification），内容被平台锁定，用户需登录才能访问，匿名性逐渐丧失。人工智能（AI）的崛起加剧了这一趋势：AI整合搜索功能，取代传统链接，导致内容创作者依赖AI优化（AIO/GEO），而用户则通过AI代理获取信息，浏览器可能演变为AI工具。文章预测，未来网络可能进一步封闭：非AI客户端被限制，AI应用内嵌广告并转向订阅模式，最终形成付费墙和阶级信息鸿沟。作者虽不乐观，但仍呼吁支持独立内容创作，如维护个人博客或捐赠维基百科，以延续开放网络的精神。
