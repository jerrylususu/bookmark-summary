# Alternatives for the EDIT tool of LLM agents - <antirez>
- URL: https://antirez.com/news/166
- Added At: 2026-05-19 14:40:32
- Tags: #read #llm #agent

## TL;DR
本文提出LLM代理中EDIT工具的两种替代方案：基于标签的编辑（使用行号和校验和，令牌效率高且可靠）和基于文件CRC32的编辑（令牌更少但可靠性较低）。作者建议通过实际使用比较，并考虑添加模式切换以灵活选择。

## Summary
本文讨论了LLM代理中EDIT工具的替代方案，旨在提高令牌效率和可靠性。作者提出了两种方法：

1. **基于标签的EDIT工具**：  
   - 使用行号和4字符标签（表示行内容的校验和），LLM通过指定行号和标签来编辑文件，避免重写旧文本。  
   - 支持单行和多行编辑，显著减少令牌使用，尤其在删除大段文本时。  
   - 优势：更高效、可靠，DeepSeek v4 Flash能有效使用此工具。  
   - 潜在改进：标签长度可调整，或包含行号以优化碰撞检测。

2. **基于文件CRC32的EDIT工具**：  
   - 使用整个文件的CRC32校验和，仅通过行号指定编辑范围。  
   - 优势：令牌更少，支持范围指定（如10:23）。  
   - 劣势：需重新计算文件CRC32，且任何无关更改都会导致编辑失败，可靠性较低。

作者建议通过实际使用比较两种方案，并考虑添加命令行切换编辑模式以灵活选择。
