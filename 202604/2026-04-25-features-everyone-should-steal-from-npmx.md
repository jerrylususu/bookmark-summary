## TL;DR
npmx.dev 是一个开源 npm 替代前端，提供安装大小、脚本披露、依赖树分析等 20 项核心功能，旨在通过竞争推动 npmjs.com 改进，并支持国际化与无障碍设计。

## Summary
npmx.dev 是一个开源的 npm 替代前端，通过竞争压力促使 npmjs.com 改进，并为包注册表网站提供了许多值得借鉴的功能。以下是其核心特性总结：

1. **安装大小显示**：展示包及其依赖的解压后总大小，而非仅压缩包大小。
2. **安装脚本披露**：在包页面显示 preinstall、install、postinstall 脚本及依赖的 npx 包，并链接到代码浏览器。
3. **过时与漏洞依赖树**：提供可展开的树状视图，标注依赖版本落后情况及 OSV 漏洞信息。
4. **版本范围解析**：显示 semver 范围对应的当前解析版本，减少 CLI 查询。
5. **模块替换建议**：对 e18e 数据集中的包提供轻量替代方案提示。
6. **模块格式与类型徽章**：显示 ESM/CJS/双格式、TypeScript 类型及 Node 引擎范围。
7. **多平台仓库统计**：从 GitHub、GitLab 等多个代码托管平台获取星标和分叉数。
8. **跨注册表可用性**：标记同时存在于 JSR 等其他注册表的包。
9. **包对比视图**：支持最多十个包的指标对比和散点图分析。
10. **版本差异**：支持浏览器内文件级版本对比。
11. **发布时间线与大小标注**：时间线显示版本发布及安装大小突变点。
12. **下载分布分析**：按版本线分解周下载量，展示生态版本分布。
13. **命令面板**：快捷键调用全局导航和页面操作，支持语义范围过滤。
14. **国际化**：支持超过 30 种语言，包括 RTL 语言。
15. **无障碍默认**：图表、视频提供详细 ARIA 标签，命令面板支持屏幕阅读器。
16. **游乐场链接提取**：从 README 提取在线代码编辑器链接。
17. **代理技能检测**：列出包含 Agent Skills 清单的包及其工具兼容性。
18. **社交功能基于 AT 协议**：使用 Bluesky 网络存储点赞、评论，数据公开可互操作。
19. **本地 CLI 管理连接**：通过本地 npm CLI 处理管理操作，无需网站登录。
20. **深色模式与自定义调色板**：支持深色主题及自定义配色。

此外，.NET 社区已推出类似项目 nugx.org，灵感来源于 npmx。
