## Summary
GitHub Copilot CLI 推出实验性功能“Rubber Duck”，通过引入不同模型家族（如 Claude 模型搭配 GPT-5.4）提供第二意见，以提升代码任务的准确性和可靠性。该功能在复杂任务中表现突出，能自动或手动触发审查，帮助发现架构错误、逻辑漏洞和跨文件冲突等问题，尤其适用于多文件、长步骤的高风险任务。用户可通过 `/experimental` 命令启用此功能。
