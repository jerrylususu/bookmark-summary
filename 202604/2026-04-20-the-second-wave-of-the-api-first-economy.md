# The Second Wave of the API-first Economy

- Added At: 2026-04-20 14:28:20
- Tags: #read

## TL;DR
文章回顾了API经济从开放乐观期、收缩期到当前由大语言模型驱动的复兴历程。第二波API以智能代理为桥梁，通过自然语言指令简化用户操作，成为服务竞争新优势，预示着API普及的新春天。

## Summary
文章回顾了API经济的发展历程，从最初的开放乐观期（第一波），经历因商业、隐私和滥用问题导致的收缩期（API寒冬），再到当前由大语言模型（LLM）驱动的复兴（第二波）。第二波的核心特征是API不再主要面向第三方开发者构建应用，而是作为服务与智能代理（AI Agent）之间的桥梁，让用户能通过自然语言指令委托代理完成任务（如转账、订餐、预订等）。这种模式更贴近产品本身功能，简化了安全模型，并可能成为服务竞争的关键优势。文章指出，尽管广告驱动或垄断型平台可能不会跟进，但已有如Basecamp、Salesforce等公司开始布局，预示着一个API广泛普及的新春天。
