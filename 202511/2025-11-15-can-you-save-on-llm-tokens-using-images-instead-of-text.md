# Can you save on LLM tokens using images instead of text?
- URL: https://pagewatch.ai/blog/post/llm-text-as-image-tokens/
- Added At: 2025-11-15 09:33:08
- Tags: #read #llm

## TL;DR
研究探索了将文本转为图像以减少大语言模型提示令牌消耗的方法。实验发现，图像输入虽能节省部分提示令牌，但多数模型会生成更多高成本的完成令牌，且处理时间翻倍，综合性价比低，实际价值有限。

## Summary
这篇文章探讨了是否可以通过将文本转换为图像来减少大语言模型（LLM）的令牌消耗。核心观点是：虽然在特定条件下可行，但综合权衡下可能并不值得。

### 实验设计
作者以安德烈·卡帕西（Karpathy）的一篇关于数字卫生的博文为素材，设计了一个简单的提示词任务，要求模型提取并总结文章中的卫生建议。实验比较了两种输入方式：
- **文本输入**：直接将博文文本作为提示词。
- **图像输入**：将博文文本转换为两张768×768像素的图像（使用高细节模式），再作为视觉提示输入。

### 关键发现
1. **提示令牌（Prompt Tokens）节省**：
   - 使用图像输入时，提示令牌数量显著减少。例如，GPT-5模型节省了超过40%的提示令牌。
   - 原因：图像可能被压缩或编码为更高效的表示，尽管原始图像文件大小远超文本。

2. **完成令牌（Completion Tokens）增加**：
   - 除GPT-5 Chat模型外，其他模型在图像输入时生成的完成令牌数量明显多于文本输入。
   - 完成令牌通常比提示令牌更昂贵，因此令牌节省的实际效益被抵消。

3. **处理效率**：
   - 图像输入的处理时间比文本输入长约一倍，响应速度较慢。

### 结论
- **技术可行性**：通过图像输入节省提示令牌是可能的，但仅限于特定模型（如GPT-5）。
- **实际价值**：由于完成令牌成本更高、处理时间延长，以及模型兼容性问题（多数模型表现不佳），整体性价比低。作者认为，除非在极少数场景下，否则这种方法的实际优势有限。
