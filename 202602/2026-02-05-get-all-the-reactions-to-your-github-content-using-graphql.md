# Get all the reactions to your GitHub content using GraphQL

- Added At: 2026-02-05 14:08:28

## TL;DR
本文介绍通过GraphQL API和GitHub CLI手动获取GitHub内容（如问题、PRs和评论）的反应，因平台无通知功能。提供查询示例，但指出限制如仅能获取近期数据、效率低下，语气幽默并批评GraphQL复杂性。

## Summary
本文介绍了如何使用GraphQL API通过GitHub CLI获取用户GitHub内容（如问题、拉取请求和评论）的反应（如表情符号点赞）。由于GitHub缺乏直接通知功能，作者通过GraphQL查询手动检索数据。文章提供了多个查询示例，包括搜索有反应的问题和PRs，以及获取评论反应，但指出查询存在限制，如只能获取最近内容、无法过滤旧反应，且评论查询效率低下。整体语气幽默，批评GraphQL的复杂性，并包含免责声明。
