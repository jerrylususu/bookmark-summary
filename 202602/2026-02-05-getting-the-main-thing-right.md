# Getting the main thing right
- URL: https://www.seangoedecke.com/getting-the-main-thing-right/
- Added At: 2026-02-05 13:34:12
- Tags: #read #career

## TL;DR
文章强调项目成功的关键是专注于核心目标（主要事情），而非次要细节，类似帕累托原则。建议通过案例分析识别主要事情，科技公司中常为交付项目。需克服不适和变化，保持灵活性，多思考焦点问题以高效实现目标。

## Summary
文章的核心观点是强调在任何项目中，尤其是科技公司环境中，成功的关键在于正确识别并专注于“主要事情”，即交付项目以实现核心目标，而非纠缠于次要细节。这类似于帕累托原则，即少数关键因素决定了大部分结果，有时甚至达到90/10或99/1的比例。如果主要事情做对了，许多小错误可以被容忍。

要识别主要事情，文章建议通过分析成功和失败案例，特别是那些出人意料的结果，来理解什么是真正重要的。在科技公司中，主要事情通常涉及交付项目以增加股东价值和让管理层满意，但这并非适用于所有生活领域。然而，许多人难以专注于主要事情，因为思考和反思可能让人不适，或者他们可能不喜欢主要事情本身，例如不喜欢商业目标。文章指出，即使不喜欢，也必须面对现实，因为忽视主要事情可能导致 burnout。

此外，主要事情可能随时间变化，例如从注重人际关系转向强调交付能力，因此需要保持灵活性，避免过度专业化，而是发展多种技能以应对环境变化。

最终，文章建议投入大量时间思考焦点问题，即使这意味着减少直接工作的时间。通过强迫自己面对核心问题，即使答案不悦，也能更好地实现目标。
