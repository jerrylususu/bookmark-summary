# 你大概不会想用 LLM 做数据分析
- URL: https://roriri.one/2026/03/05/llm-stat
- Added At: 2026-03-06 14:16:42
- Tags: #read #llm

## TL;DR
该文指出LLM因训练数据含统计错误且缺乏推理能力，在数据分析中易产生误导结果，强调需懂统计原理并亲自验证，仅在可视化等辅助场景谨慎使用。

## Summary
文章指出，用大语言模型（LLM）进行数据分析存在严重风险，主要源于模型本身基于包含大量统计错误的学术文献训练，且缺乏真正的统计推理能力。作者通过实例和研究说明，LLM容易在不当提示下生成看似专业但实际错误的分析结果，例如自动化p-hacking或错误解释统计显著性。因此，作者强调两条核心法则：第零法则——若不懂统计方法就不应使用；规则一——必须理解模型行为，亲自验证代码和方法。安全用法包括可视化、聚类分析和编码辅助，但需保持专业判断，避免依赖模型作为黑箱。
