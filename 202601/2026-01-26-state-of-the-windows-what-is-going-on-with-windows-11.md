# State of the Windows: What is going on with Windows 11?

## TL;DR
文章批评Windows 11自2023年质量下降，更新频发错误、系统臃肿运行慢、AI功能强制集成但实用性低，微软被指忽视用户体验，呼吁回归稳定可靠系统。

## Summary
文章探讨了Windows 11的当前状态，指出自2023年以来系统质量显著下降。主要内容包括：

- **近期更新问题**：2026年1月更新引入多个严重错误，如PC无法正常关机（由KB5073455引起，已通过KB5077797修复）、Outlook无法打开PST文件（由KB5074109引起，已通过KB5078132修复），以及其他问题如应用崩溃和系统无法启动。
  
- **系统性问题**：Windows变得臃肿，核心组件如Windows Explorer运行缓慢，更新体积大幅增加（如2026年1月更新是之前版本的四倍大），且技术债务积累严重，导致稳定性差。

- **AI集成批评**：微软过度强调AI功能，如Windows Recall存在安全漏洞（数据存储不安全，后经修复），Copilot被强制集成到多个应用（如Edge、Notepad、Settings等），但多数功能不实用或用户不需要。同时，本地账户设置越来越困难，系统倾向于在线账户。

- **整体评价**：Windows 11被视为一个基础不稳定的系统，微软优先追求AI创新而忽略了用户体验和可靠性。作者呼吁微软应将重点回归到提供稳定、可靠的操作系统。
